Sobre

Esta página foi desenvolvida como exercício prático para consolidar conceitos de estrutura semântica em HTML.

O foco está na organização correta dos elementos e no uso adequado de marcações como títulos hierárquicos e seções bem definidas.

Conteúdo

  • Uso correto de tags semânticas
  • Hierarquia de títulos
  • Navegação com âncoras internas
  • Formulários acessíveis
