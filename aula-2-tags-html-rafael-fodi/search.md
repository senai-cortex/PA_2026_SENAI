1. `<html>`  
Breve descrição: Elemento raiz do documento.  
Para que serve: Envolve todo o conteúdo da página HTML.  
- `lang`: idioma do conteúdo, como "pt-BR" ou "en".  
- `dir`: direção do texto, "ltr" ou "rtl".  
- `xmlns`: namespace do documento XHTML.  

2. `<head>`  
Breve descrição: Armazena metadados.  
Para que serve: Configurar informações que não aparecem visualmente.  

3. `<title>`  
Breve descrição: Título da aba do navegador.  
Para que serve: Identificar a página no navegador e em buscadores.  

4. `<meta>`  
Breve descrição: Define metadados adicionais.  
Para que serve: Especificar charset, descrição, viewport etc.  
- `charset`: codificação, geralmente "UTF-8".  
- `name`: tipo de informação ("description", "viewport").  
- `content`: valor associado ao `name`.  

5. `<body>`  
Breve descrição: Área do conteúdo visível.  
Para que serve: Abrigar tudo que o usuário enxerga.  

6. `<h1>` a `<h6>`  
Breve descrição: Títulos hierárquicos.  
Para que serve: Organizar o conteúdo por níveis de importância.  

7. `<p>`  
Breve descrição: Parágrafo de texto.  
Para que serve: Separar blocos textuais.  

8. `<a>`  
Breve descrição: Hiperlink.  
Para que serve: Criar navegação interna ou externa.  
- `href`: destino do link.  
- `target`: onde abrir o link.  
- `title`: informação adicional ao passar o mouse.  

9. `<img>`  
Breve descrição: Insere imagem.  
Para que serve: Exibir conteúdo visual na página.  
- `src`: caminho da imagem.  
- `alt`: descrição alternativa obrigatória.  
- `width` e `height`: definem dimensões.  

10. `<ul>`  
Breve descrição: Lista não ordenada.  
Para que serve: Criar lista com marcadores.  

11. `<ol>`  
Breve descrição: Lista ordenada.  
Para que serve: Criar lista numerada.  
- `type`: formato da numeração.  
- `start`: número inicial.  
- `reversed`: inverte a ordem.  

12. `<li>`  
Breve descrição: Item de lista.  
Para que serve: Representar cada elemento da lista.  

13. `<div>`  
Breve descrição: Container genérico em bloco.  
Para que serve: Agrupar elementos estruturalmente.  

14. `<span>`  
Breve descrição: Container genérico inline.  
Para que serve: Agrupar trechos pequenos de texto.  

15. `<header>`  
Breve descrição: Cabeçalho semântico.  
Para que serve: Representar parte introdutória da página ou seção.  

16. `<nav>`  
Breve descrição: Área de navegação.  
Para que serve: Agrupar links principais.  

17. `<section>`  
Breve descrição: Seção temática.  
Para que serve: Dividir conteúdo por assunto.  

18. `<article>`  
Breve descrição: Conteúdo independente.  
Para que serve: Representar postagem, notícia ou conteúdo autônomo.  

19. `<table>`  
Breve descrição: Estrutura de dados tabulares.  
Para que serve: Organizar dados em linhas e colunas.  
- `border`: define borda simples.  
- `cellpadding`: espaço interno da célula.  
- `cellspacing`: espaço entre células.  

20. `<tr>`  
Breve descrição: Linha da tabela.  
Para que serve: Agrupar células horizontalmente.  

21. `<td>`  
Breve descrição: Célula de dados.  
Para que serve: Inserir valores na tabela.  
- `colspan`: ocupa múltiplas colunas.  
- `rowspan`: ocupa múltiplas linhas.  
- `headers`: associa a cabeçalhos.  

22. `<th>`  
Breve descrição: Célula de cabeçalho.  
Para que serve: Identificar títulos de colunas ou linhas.  
- `scope`: define escopo ("row" ou "col").  
- `colspan`: mescla colunas.  
- `rowspan`: mescla linhas.  

23. `<form>`  
Breve descrição: Estrutura de formulário.  
Para que serve: Coletar e enviar dados do usuário.  
- `action`: destino do envio.  
- `method`: método HTTP.  
- `autocomplete`: ativa/desativa preenchimento automático.  

24. `<label>`  
Breve descrição: Rótulo de campo.  
Para que serve: Associar texto ao campo do formulário.  
- `for`: conecta ao `id` do campo.  

25. `<input>`  
Breve descrição: Campo de entrada.  
Para que serve: Permitir inserção de dados.  
- `type`: tipo do campo.  
- `name`: nome enviado ao servidor.  
- `required`: torna obrigatório.  

26. `<textarea>`  
Breve descrição: Campo de texto multilinha.  
Para que serve: Inserir textos maiores.  
- `name`: nome do campo.  
- `rows`: quantidade de linhas.  
- `cols`: largura em colunas.  

27. `<button>`  
Breve descrição: Botão clicável.  
Para que serve: Executar ação ou enviar formulário.  
- `type`: tipo do botão.  
- `name`: identificação no envio.  
- `value`: valor associado.  

28. `<footer>`  
Breve descrição: Rodapé semântico.  
Para que serve: Incluir informações finais da página ou seção.  

29. `<main>`  
Breve descrição: Conteúdo principal.  
Para que serve: Delimitar a parte central e única da página.  

30. `<br>`  
Breve descrição: Quebra de linha.  
Para que serve: Inserir nova linha sem criar parágrafo.
